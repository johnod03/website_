Game Instructions

Luanch a clay pigeon by pressing the right hand button labeled 'PULL'.

Try to hit the clay pigeon by targeting with your mouse or tapping it on your mobile device screen.

Reload by Hitting the 'New Clay Pigeon' Button on the left hand side.

Enjoy!

Flash HTML 5 Development

As part of the Multimedia workshops, ran by Dr. Mikael Fernstrom, we were required to explore the world of HTML5 canvas. This allows the creation of interactive artefacts that can be hosted on the web.

Of all the workshop components, I found this to be the most challanging.

I can , however, see the power of this approach to internet interactivity. By using, what used to be called Flash by Adobe, now simply called Animate, it is possible to create complex and intricate animations, program their behaviours using Actionscript 3 and convert the assets to a web friendly form using the create.js javascript library.

Basic Game

The basic game hosted on this page uses the software suites mentioned. I would have liked to given myself more time delve into these technologies and create a much more finished game with scores, levels and a lot more interactive elements. The limits seem endless with this approach to programming for the web. And I feel like I haven't even begun!

I would like to continue the development of this game and start exploring the possibilities of working with sound within the HTML5 Canvas tag. A small sample of sounds are used in the game. The clay pigeon has a bullet sound and some ambient forest/ bird song plays in the background.
