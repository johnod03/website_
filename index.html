ABOUT ME

This site documents the work that was created while undertaking the Interactive Media Masters program at University of Limerick, Ireland.

I received an undergraduate honors degree in Creative Multimedia from Limerick School of Art & Design in 2015. I am a musician and play bass guitar in Supersoul Machine full time.

iMedia

This website was coded using HTML 5, jQuery, javascript and the Twitter bootstrap plugin. Bootstrap was used to create a responsive, mobile friendly design while javascript and jQuery were used to implement some animations, effects and the image gallery on the photography page. A bootstrap template was customised to get the look I was after, particularily with the dropdown menu toggle button that appears on mobile devices.

This website is still under development and will remain so until the end of the Masters course in August 2016. There are still some improvements to be made, especially relating to, visual design, information architecture and interactions. Media quieries have been implemented to allow the site to adapt to different screen sizes. Until now the site has been tested for different screen sizes by resizing the browser window. This is not a perfect way to test for screen size and responsivness. So, this will be the main improvement that will be dealt with in the next iteration of this website.
